Integrating Sprite JS

The Sprite JS Library consists of one file: sprite.js. Though you may wish to use it's minified version here. Either way, download one of the copies and include it in your website's source files.

Once downloaded, embed the Sprite JS Library into your html. For example...

<script src='js/sprite.js'></script>
