Introduction

Welcome!

Welcome to Sprite JS! Creating an Sprite JS application is easy, and only takes a few minutes.

In this tutorial, you will learn how to:

  • build a Sprite JS application using Javascript
  • create textures, scenes, and sprites
  • define mouse regions in the canvas
  • create your first Sprite JS program
  • have fun!

By the end of the tutorial, you will have implemented a working application, a simple pacman game.

Next

To get started developing Sprite JS applications, you download and set up the Sprite JS Library.
